cuDOLFINx documentation
=======================

cuDOLFINx is an extension to the DOLFINx package in the `FEniCS
Project <https://fenicsproject.org/>`_ that provides CUDA-accelerated assembly routines. It is developed on `GitHub
<https://github.com/bpachev/cuda-dolfinx>`_.
